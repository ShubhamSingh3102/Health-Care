# Health-Care

# 🏥 Healthcare Management System

A modern healthcare management platform built using Python that helps patients and doctors connect efficiently. The system provides features like appointment booking, health tracking, video consultations, and secure user management.

---

## 🚀 Features

* 👨‍⚕️ Doctor Appointment Booking
* 🔐 Secure User Authentication
* 📹 Video Consultation Support
* ❤️ Health Tracking Dashboard
* 📂 Patient Record Management
* 📊 Interactive User Interface
* 🗄️ Database Integration

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Database:** SQLite
* **APIs:** Integrated Healthcare APIs

---

## 📦 Installation

```bash
# Clone the repository
git clone <your-repository-link>

# Move into project folder
cd healthcare-management-system

# Install dependencies
pip install -r requirements.txt

# Run the project
streamlit run app.py
```
